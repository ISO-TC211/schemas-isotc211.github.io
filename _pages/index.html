ISO/TC 211 Schemas

Schema Repository

The authoritative repository of XML and JSON schema representations for ISO/TC 211 Geographic information/Geomatics standards. Browse interactive documentation for each schema, or download the files directly.

XML Schema location URL pattern

https://schemas.isotc211.org/{Standard}/{Part}/{Prefix}/{Version}

JSON Schema location URL pattern

https://schemas.isotc211.org/json/{Standard}/{Part}/{Module}/{Version}

Resources

Supporting resources for ISO/TC 211 implementers: codelist definitions, example instances, validation rules, and transform stylesheets.

Codelist Catalogues

70 canonical codelist definitions for all TC 211 standards, with browseable HTML pages and stable dereferenceable URIs.
